# Features and Workflows

Complete guide to Olanma features and how to use them.

## Core Features

### Chat and Conversations

Create conversations and chat with AI models.

Features:
- Multiple conversations per user
- Message streaming with real-time display
- Support for multiple LLM providers and models
- Message history and context
- Pin/unpin conversations
- Soft delete with recovery option

Workflow:
1. Create new conversation
2. Select LLM model
3. Type message
4. LLM responds with streaming tokens
5. View full response and continue conversation

### Document Processing

Upload and process documents for context-aware answers.

Supported Formats:
- PDF (with text extraction)
- Plain Text
- Word documents (coming soon)

Processing:
1. Upload file (automatic validation)
2. File stored on server with status "pending"
3. Background job extracts text/content
4. Summary generated by LLM
5. Document ready for use in chat

Use in Chat:
1. Click attachment button in message input
2. Select document(s)
3. Send message with document context
4. LLM includes document content in response

### Audio Processing

Upload audio files for transcription.

Supported Formats:
- MP3
- WAV
- M4A
- FLAC

Processing:
1. Upload audio file
2. Backend extracts audio (if from video)
3. OpenAI Whisper transcribes audio
4. Transcript stored and searchable
5. Can reference in chat

### Video Processing

Upload videos for analysis and transcription.

Supported Formats:
- MP4
- MOV
- AVI

Processing:
1. Upload video file
2. Extract audio stream
3. Transcribe audio to text
4. Generate summary using LLM
5. Extract action items
6. Store all metadata

### Retrieval-Augmented Generation (RAG)

Ask questions about your documents with automatic context.

How RAG Works:
1. User asks question
2. Question converted to embedding (vector)
3. Vector database searched for similar content
4. Top results retrieved and added to context
5. LLM uses context + question to generate answer
6. Response more accurate and grounded

Enable RAG:
```
When sending message, check "Use RAG"
or include documents in message
```

### Embeddings and Vector Search

Semantic search capability for documents.

Features:
- Automatic embedding generation
- Semantic similarity search
- Configurable embedding models
- Vector database storage

### LLM Provider Management

Support for multiple LLM providers.

Supported Providers:
- OpenAI (GPT-4, GPT-3.5-Turbo)
- Anthropic (Claude models)
- Cohere
- HuggingFace

Features:
- Add multiple providers
- Switch between providers per conversation
- Fallback to alternative provider
- API key encryption and security

### Workspaces and Collaboration

Organize conversations and collaborate with others.

Features:
- Create workspaces for projects
- Invite collaborators with role-based access
- Share conversations within workspace
- Workspace-level permissions
- Activity tracking

Roles:
- Owner: Full control
- Editor: Create and edit conversations
- Viewer: Read-only access

### Recent Activities

Track what you've done and recent changes.

Activity Types:
- Conversation created
- Message sent
- Document uploaded
- Collaborator added
- Conversation deleted

Access:
- View recent activities in sidebar
- Filter by type and date
- Search activity history

## Advanced Workflows

### Analyzing Multiple Documents

Workflow:
1. Upload all documents (PDF, audio, video)
2. Create conversation
3. In message: "Analyze all documents I uploaded"
4. Attach all documents
5. LLM analyzes all documents together
6. Get comprehensive analysis

### Creating Research Collections

Workflow:
1. Create workspace for research
2. Upload all source materials
3. Start conversations for each topic
4. Reference materials across conversations
5. Build knowledge base
6. Invite collaborators to review

### Building a Knowledge Base

Workflow:
1. Upload all company documents
2. Generate embeddings (automatic)
3. Use RAG to query knowledge base
4. Answers automatically include sources
5. Keep documents updated
6. Share with team

### Batch Processing

Using background jobs for bulk operations.

Features:
- Process multiple documents
- Generate embeddings in batch
- Schedule tasks
- Monitor job status
- Error handling and retries

### Custom Integrations

```python
import requests

# Authenticate
response = requests.post(
    'https://your-olanma.com/api/v1/auth/login',
    json={
        'email': 'user@example.com',
        'password': 'password'
    }
)
token = response.json()['access_token']

# Create conversation
headers = {'Authorization': f'Bearer {token}'}
response = requests.post(
    'https://your-olanma.com/api/v1/chat/conversations',
    headers=headers,
    json={'title': 'Integration Test'}
)
conversation_id = response.json()['id']

# Send message
response = requests.post(
    'https://your-olanma.com/api/v1/chat/stream',
    headers=headers,
    json={
        'conversation_id': conversation_id,
        'content': 'Hello from integration'
    }
)
```
